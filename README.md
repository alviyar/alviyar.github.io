# alviyar.github.io

Personal portfolio and project landing page for Alviya Rizwan.

Live at: https://alviyar.github.io

---

## Stack

Built with plain HTML, CSS, and vanilla JavaScript.
No frameworks, no build step, no client-side routing.

## Structure

- `index.html` — all sections (hero, about, skills, projects, experience, contact)
- `styles.css`  — dark theme, CSS variables, responsive grid
- `app.js`      — project filter, scroll-spy nav, fade-in animations
